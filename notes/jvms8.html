Ссылочные типы и значения

Есть три типа ссылочных типов: 1) class types 2) array types 3) interface types

Их значениями являются ссылки на динамически созданные: 1) экземпляры классов 2) экземпляры массивов 3) экземпляры классов или массивы реализующие соответствующий интерфейс соответственно.

Тип массивов содержит так называемый component type (тип элементов массива). Он может быть любым ссылочным или примитивным типом, в том числе и массивом. Значение ссылочного типа может быть специальная нулевая ссылка, null. Нулевая ссылка быть присвоена если значение не определено в рантайме, но оно может быть приведено к любому типу. Значение по умолчанию для ссылочного типа - null. Эта спецификация не регламентирует конкретное значение для кодирования null.
